With the help from my friends at Turnstyle, a local design agency, I was able to work on several projects using their design comps, starting with Fulcrum. I was very excited to start working on Fulcrum since it was my first website using a design comp as a guide and my first website using jQuery.

Lessons Learned

  • jQuery
  • Plugins

My biggest lesson learned was just how much you can do with jQuery and how to use it. When I started off, I got lost in jQuery’s API trying to make sense of all the different functions it provides. I studied from both Treehouse and CodeCademy to help further my understanding of how to call functions and target elements from my markup. Fulcrum was probably the longest project to complete since I was experimenting a lot with it.

Another lesson learned was acknowledging that sometimes using a plugin can save you a lot of time instead of building the functionality from scratch (especially when you’re a jQuery n00b like I was). I spent hours trying to build the second level menu transition shown below. I read countless tutorials, spent so much time on jQuery’s API, and even browsed some plugins I could use. When I finally got my menu transitions to work, I was beyond excited. However, I realized that if I was on a deadline and needed to get things done quickly, building out everything from scratch might could take up precious time.

Either way, I'm still pretty happy with how it turned out but I did use a plugin called Owl Carousel for the spotlight section.
